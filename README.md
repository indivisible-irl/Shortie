Shortie
=======

Android URL shortener service

Integrates with the "Share" context for any valid URL. Returns a short version in clipboard for easy use in any situation.


Versions
========

0.0.1    (2012/09/11):
    Initial up.
    Shell functionality, listeners but no actions.
